How it works

Trip Brief Agent 先把你的句子變成結構化需求；Timezone / Local Discovery（帶網路搜尋）並行做功課；Composer 合成逐日行程；渲染器印出封面與每日票券。

真實模式全程約 5–8 分鐘（網路搜尋佔大半）。過程中每個 agent 的狀態都會誠實記錄在手冊裡。

Gmail / Calendar agent 尚未接上 connector，會標記為 skipped。
